CUDA MSA

Improved Center Star Algorithm for Multiple Sequences Alignment Based on CUDA

Download as .zip Download as .tar.gz View on GitHub

Install

1. Download the source code

2. Run the ./build script in the folder

Usage

./msa.out [options] input_path output_path

Options:

-g : use GPU only (default use both GPU and CPU) -c : use CPU only (default use both GPU and CPU) -w <int> : specify the workload ratio of CPU / CPU -b <int> : specify the number of blocks per grid -t <int> : specify the number of threads per block
